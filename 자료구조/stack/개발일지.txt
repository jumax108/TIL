21/08/11 변경점

1. 테스트 코드 추가
코드 안정성을 확보하기 위해 테스트 코드를 작성했습니다.
코드가 변경되면 동일한 테스트 코드를 통해 최소한의 안정성을 확보할 예정입니다.

2. 반환되는 값의 의미 통일
기존에는 push 함수는 push 성공 여부를 반환했습니다.
하지만 front, getSize, getCapacity는 각 함수에 요청한 데이터를 반환하고 있었습니다.
이것을 통일하여 모든 함수는 함수 성공 여부를 반환하도록 하였고,
포인터를 인자로 받아 데이터를 전달하도록 수정했습니다.
반환되는 타입이 void 인 경우는 실패하는 경우가 없는 함수입니다.

3. 네임스페이스 제거
기존 STL stack과의 구분을 위해 my 네임스페이스를 적용하였습니다.
제거한 이유는 두가지 입니다.
첫째, 굳이 stl 스택과 같이 쓸 일이 없을 것 같다.
둘째, 클래스 이름을 CStack으로 변경하여서 이름이 충돌하지 않는다.
